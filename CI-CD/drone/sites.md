Drone 是一个持续交付平台，可帮助您的组织优化和自动化软件交付。

dron 是一种基于容器技术的持续交付系统。Drone 使用简单的 YAML 配置文件（docker-compose的超集），来定义和在 Docker 容器中执行 Pipelines。

---

plugins 是 Docker 容器，用于执行预定义的任务，并配置为 pipeline 中的步骤。插件可用于部署代码，发布工件，发送通知等。
